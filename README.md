OPSBroadcastEmail
=================

Intelligent Email Messaging Solution

Ozeki Broadcast By Email integrates an intelligent, automated email sending solution into your communication system. After you have sent your customized text emails, you can easily track the status of your messages. This way, you can manage your email traffic efficiently.

Quick Start Tips
=================
1. Specify your API Extension ID: Click on the 'Settings' tab then enter the ID of your API Extension that has been added previously in Ozeki Phone System XE
2. Import an email list: Create a CSV file filled out the table with user data (RecipientAddress, EmailSubject, EmailContent) then import your existing mail list.
3. Edit the parameters: If you need you can easily modify the subject and the text in each line.
4. Send your email: Click on 'Start' button to send your email.
5. Monitor the status of the sent emails: Check what percentage of your messages were delivered and undelivered. You can see the messages in progress as well.
